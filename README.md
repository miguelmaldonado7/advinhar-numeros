Este é o meu primeiro projeto prático desenvolvido em C++! É um jogo clássico de adivinhação pelo terminal, construído para treinar lógica de programação. 

O projeto foi proposto como um desafio do **roadmap.sh**. Você pode conferir os detalhes da tarefa original aqui:
[Acessar o desafio no roadmap.sh](https://roadmap.sh/projects/number-guessing-game)

O jogo funciona gerando um número aleatório que o jogador precisa descobrir. A cada tentativa, o programa dá dicas se o palpite foi muito alto ou muito baixo. 

Ao desenvolver este código, pude colocar em prática conceitos fundamentais do C++, como:
* Variáveis e tipos de dados.
* Estruturas condicionais (`if` / `else`).
* Laços de repetição (`loops`).
* Entrada e saída de dados no console do Windows.
